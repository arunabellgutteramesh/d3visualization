Visualization of Suicide Causes in India

(for the period from 2003 to 2012)




Graph 1 : Stacked Bar Graph

Shows the number of deaths versus causes for the years from 2003 to 2012. The stack shows the comparison between the number of deaths in each sex per category (here cause).

Graph 2 : Multiple Line Graph

The graph below shows top 5 causes/reasons for suicide in India, segregated by age groups, over the period from 2003 to 2012.
Select an age group from the dropdown to generate a graph. By varying the age group selection one can study which age group has more tendency to commit suicide over the other age groups along with the reasons that predominates that age group.

Graph 3 : Map - India map with state borders

The map below shows the total number deaths due to suicide per state per year from the years 2003 to 2012. Hover over the state to get the state name with number of deaths in that state.
Select a year from the dropdown to get the desired map of that year. By varying the year, one can understand which state falls under which bandwidth (shown in scale of map) for that year.
